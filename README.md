# bot-storybook

> A GitHub App built with [Probot](https://github.com/probot/probot) that comments PRs with a link to storybook when deployed by CI.

## Setup

```sh
# Install dependencies
npm install

# Run typescript
npm run build

# Run the bot
npm start
```
